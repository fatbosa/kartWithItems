Instruções de build: Ao importar o package, clique em file>build settings. Após isso, arraste as cenas para a nova tela na seguinte ordem: MainMenu como index 0, Phase1 como
index 1 e Phase2 como index 2. Depois disso já dá pra fazer a build.

Para jogar, controla-se o personagem com W para acelerar, A e D para mover pros lados, E para usar itens e a barra de espaço para pular.
Para voltar ao menu basta apertar Esc.
